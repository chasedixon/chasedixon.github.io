Project Description

A number guessing game, where a registered user tries to guess a random number between 1 and 100. Following each guess, the user gets a hint whether the number is higher or lower than the guess. When guessed correctly, the user's score is added the the High Scores table.

Frameworks/Languages Utilized

  • InfinityFree Web Hosting
  • PHP
  • MySQL

Challenges:

This was my first experience with php, so my main challenge was learning the syntax and switching back and forth between php and html. I really liked how easy it was to implement, and having the ability to add logic straight into the html.
