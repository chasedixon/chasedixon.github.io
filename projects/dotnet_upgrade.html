Project Description

Worked in a team to upgrade an existing .NET 2.0 web application to .NET Core 6.0. Performed multiple tests of application to find and fix numerous bugs.

Frameworks/Languages Utilized

  • Entity Framework (MySQL)
  • .NET Core (C#)
  • Razor Pages

My Contributions

I performed the first migration from 2.0 to 2.1, then assisted a teammate in completing the migration to 2.2. I also found and fixed multiple bugs, including one restricting the automatic creation of a missing database and multiple bugs causing pages to crash/load incorrectly.

Challenges:

Upgrading through the different versions proved to present many challenges. At first, we followed along with Microsoft's migration documents. After the first two migrations we discovered the migration assistant that helped with the bulk of the final migrations, but missed many of the build errors that we had to fix manually.
